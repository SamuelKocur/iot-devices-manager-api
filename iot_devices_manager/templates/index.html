IoT Devices Manager - Bachelor thesis

Samuel Kocúr

The aim of the bachelor thesis was to create a system for managing IoT device and analyzing collected data. The project consists of three parts: server, mobile app and web app. The backend and web administration components of the system were built using Django and Django Admin. Mobile app was developed in Flutter and main purpose of the app is tracking available IoT devices and analyzing data.
